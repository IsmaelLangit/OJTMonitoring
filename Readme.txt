 * INTRODUCTION

 - OJT Monitoring program, for monitoring OJT 2 students' company,
   status of requirements, etc.. 

---------------------------------------

 * REQUIREMENTS

 - WampServer (Download the WAMP version that suits your OS 
   e.g. if your OS is 64 bit, download that version.)

 - Text Editor (Notepad++, Brackets, or Sublime) 

 - Web Browser	

---------------------------------------

 * INSTALLATION

 1. Run Wamp server 
 2. Open phpMyAdmin by clicking the phpMyAdmin icon
 3. Create New Database 
 4. Type 'ojt2monitoring' and click create button
 5. import the database
 6. Go to driver you installed wamp on \wamp64\www\.
 7. Copy the 'ojtmonitoring' folder on \wamp64\www\.
 8. Open your browser
 9. Type 'localhost/ojtmonitoring'


---------------------------------------

 * FUNCTIONALITIES
 
 Functional:
 + Display ID no., Name, Course & Year, Company Name, Status
 + Tools: Edit student information, Delete Student
 + Filter students by status (e.g. partial, complete, incomplete)
 + Filtering of Student Name
 + Add Student/s
 + Viewing of student information/profile

 Non-Functional
 - 'sample1' heading and contents 
 - show 'n' entries 
 - previous button and next button
